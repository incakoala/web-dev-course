The Amazon Physical Stores Team is developing a way to quickly redesign the layout of the displays in its Amazon 4-Star stores. The floor plan configuration and a display unit are represented as two strings s and t, respectively. Create a function that performs the layout redesign per the following rules. 

  • Check whether string s is divisible by string t. A string s is divisible by string t if string t can be concatenated some number of times to obtain the string s.
  • If s is divisible, find the smallest string u such that it can be concatenated some number of times to obtain both s and t.


If it is not divisible, set the return value to -1.

Write an algorithm to find the length of string u if string s is divisible by string t else output -1.

Input
The input to the function/method consists of two arguments:
planConfig, a string representing the  floor plan configuration (s);
displayStr, a string representing the display unit layout (t).

Output
Return an integer representing the length of the smallest string which divided both the strings planConfig and displayStr, if string planConfig is divisible by string displayStr. If no such string is possible then return -1.

Constraints
1 ≤ length of planConfig ≤ 2*105
1 ≤ length of displayStr ≤ 2 *105
length of displayStr ≤ length of planConfig

Examples
Example 1:
Input:
planConfig = "bcdbcdbcdbcd"
displayStr = "bcdbcd"

Output:
3

Explanation:
If string "bcdbcd" is concatenated twice, the result is "bcdbcdbcdbcd" which is equal to the string planConfig. The string planConfig is divisible by string displayStr. 
Since it passes the first test, look for the smallest string that can be concatenated to create both strings planConfig and displayStr.
The string "bcd" is the smallest string that can be concatenated to create both strings. 
The length of the string is 3.
So the output is 3.
